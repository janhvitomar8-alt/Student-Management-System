# Student Management System

A command-line Student Management System built using Python.

## Features

* Add Student
* View Students
* Search Student
* Delete Student
* Save Student Records
* Load Student Records Automatically

## Concepts Used

* Python Functions
* Lists
* Dictionaries
* Loops
* Conditional Statements
* File Handling
* User Input

## How to Run

1. Install Python.
2. Open a terminal in the project folder.
3. Run:

python student_management.py
